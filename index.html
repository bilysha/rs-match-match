Welcome to the match-match game.

about :

Memory is a counter game where the object is to find pairs. When the game begins, all pictures are hidden.

Rules :

1. Select two cards to try to match the pictures.

2. If you match the pictures, they will be disappear and you can go again.

3. If they don't match, you also may select two cards to try to match the pictures again :)

4. When all the cards disappear, you will win !

5. Have Fun !
